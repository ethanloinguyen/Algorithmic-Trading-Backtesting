── How to Run Locally ───────────────────────────────────────────────────────────
 
Backend:
    cd backend
    pip install -r requirements.txt
    python run.py              # http://localhost:4000
    # Docs at http://localhost:4000/docs
 
Frontend:
    cd Frontend
    npm install
    npm run dev                # http://localhost:3000
